## 待开发

### ⏳ Batch 3：视觉打击增强

> **核心交付**：击杀有"爆破感"——闪光→膨胀环→碎片散开，伤害数字弹出来

#### 改动文件

| 文件 | 改动 |
|------|------|
| `src/VFX.ts` | 3 层爆炸 + 伤害数字入场动画 + 方向性粒子 + 元素颜色体系 |
| `src/config.ts` | +ELEMENT_COLORS + DAMAGE_NUMBER_CONFIG |

#### 具体实现

**3 层爆炸系统**（参照 Soulstone Survivors）

| 层 | 内容 | 时长 | 缓动 |
|----|------|------|------|
| 内层闪光 | 白色精灵，alpha 1→0 | 50ms | Linear |
| 膨胀环 | 环精灵 scale 0.5→2.0，alpha 0.8→0 | 200ms | Power2 |
| 抛射粒子 | 10-20 个彩色粒子，随机方向，alpha 衰减 | 400ms | Power1 |

应用于：技能命中 + 怪物死亡

**伤害数字入场动画**

```
弹入阶段（150ms）：scale 0.3 → 1.2 → 1.0，Back.easeOut
上飘阶段（550ms）：y -36px，alpha 1→0，Power1
暴击：初始 scale ×2，红色，粗体
偏移：X 方向随机 ±8px 防止重叠
```

**方向性粒子**（参照 Vlambeer Nuclear Throne）

`VFX.burst()` 增加 `directionAngle` + `spreadAngle` 参数。
命中时粒子沿攻击方向 ±45° 喷射，死亡时保持 360° 随机。

**元素颜色映射**（参照 Death Must Die 9 神体系）

| 元素 | 命中色 | 粒子色 | 震屏 |
|------|--------|--------|------|
| physical（木梁/碎石）| #FFFFFF | 灰/棕 | light |
| fire（彩绘爆炸）| #FF6600 | 红/橙/黄 | medium |
| ice（防水水纹）| #00CCFF | 青/白/蓝 | light |
| nature（药雾）| #44AA00 | 绿系 | medium |

#### 体验验收

- [ ] 击杀怪物：白色闪光 → 环向外扩散 → 彩色碎片粒子散开飘落
- [ ] 伤害数字：从小到大"弹"出来再上飘消失
- [ ] 粒子方向：命中时往攻击方向喷射，死亡时 360° 散开
- [ ] 颜色：木梁棕色 / 防水蓝色 / 彩绘彩色
- [ ] 大量击杀时粒子不拖慢游戏

### ⏳ Batch 4：UI 反馈 + 技能展现

> **核心交付**：技能冷却 UI、投射物拖尾、镜头冲击、怪物生成特效

#### 改动文件

| 文件 | 改动 |
|------|------|
| `src/HUD.ts` | 技能冷却指示器（5 个圆） |
| `src/SkillManager.ts` | 投射物拖尾粒子 |
| `src/GameScene.ts` | 自动攻击冷却环 + 镜头冲击 + 怪物生成特效 |
| `src/config.ts` | +CAMERA_PUNCH_CONFIG |

#### 具体实现

**技能冷却 HUD**（参照 Death Must Die 恩赐 UI）

右侧 5 个圆形技能图标：
- 每个槽 14px 圆 + 扇形冷却覆盖（Graphics arc，从顶部顺时针收缩）
- 就绪时绿色脉冲 + `skillReady()` 音效
- 5 个图标垂直排列，间距 44px

**投射物拖尾**（参照 20 Minutes Till Dawn）

每 30ms 在投射物身后留渐隐小圆点：
- 木梁 → 棕色 #C4884D
- 颜料弹 → 彩虹粒子随机色
- 自动箭矢 → 浅蓝 #88CCFF

**自动攻击冷却环**

玩家周围 22px 半径圆环，随时间填充（逆时针），满时白色脉冲。

**镜头冲击**（参照 Vlambeer Nuclear Throne）

重击时 camera 快速 zoom in 再恢复：
- heavy: zoomTo(1.04, 40ms) → zoomTo(1.0, 200ms)
- ultra: zoomTo(1.06, 50ms) → zoomTo(1.0, 300ms)

**怪物生成特效**

alpha 0 + scale 0.3 → tween 到 alpha 1 + scale 1（200ms Back.easeOut）+ 5 个粒子爆发。

- [ ] 重击时镜头短暂放大弹回

### ⏳ Batch 5：打磨收尾

> **核心交付**：灼烧视觉、平滑血条、玩家移动手感

#### 改动文件

| 文件 | 改动 |
|------|------|
| `src/Building.ts` | 灼烧视觉叠加 |
| `src/HUD.ts` | HP/EXP 平滑条 lerp |
| `src/Player.ts` | 加速减速 + 无敌闪烁 |
| `src/config.ts` | +AUTO_ATTACK_CONFIG（移动硬编码值） |

#### 具体实现

**灼烧视觉**

`applyBurn()` 叠加橙色闪烁圆（alpha 0.1↔0.35 yoyo），灼烧期间持续。

**平滑血条**

```
每帧 lerp: displayHp += (actualHp - displayHp) * 0.08
          displayExp += (actualExp - displayExp) * 0.10
```

**玩家移动微调**

- 加速：`velocityX += inputX * 1500 * dt`
- 减速：`velocityX = moveToward(velocityX, 0, 2000 * dt)`
- 无敌闪烁：被击退后 1s 内 4Hz 闪烁（alpha 1.0 ↔ 0.4）

#### 体验验收

- [ ] 火焰怪打古建时有橙色光圈闪烁
- [ ] 血条不是瞬间跳变而是平滑滑动
- [ ] 经验条也是平滑增长
- [ ] WASD 有轻微加速感，松键不会瞬间急停
- [ ] 被击退后短暂闪烁
