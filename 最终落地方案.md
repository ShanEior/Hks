## Phase 3：Hit Stop 单怪冻结（~35 行，45min）

### 改什么

| 文件 | 改动 |
|------|------|
| `config.ts` | `HIT_STOP_CONFIG` 新增 `none` 档 (freezeMs:0) |
| `Monster.ts` | 新增 `freezeTimer` 字段；`update()` 中冻结时跳过 AI；`takeDamage()` 中按伤害设置 freezeTimer |
| `GameScene.ts` | 普攻命中传入 tier='none' |
| `CombatFeel.ts` | 去除 time control 职责，仅保留 VFX + 镜头冲击 |

### 新的 Hit Stop 分级

| 等级 | 冻结 | 适用 |
|------|------|------|
| none | 0ms | 普攻 |
| light | 33ms | DOt/溅射 |
| medium | 50ms | 普通技能 |
| heavy | 83ms | 暴击/闪电链 |
| ultra | 150ms | Boss死亡/出场 |

### 验收标准

- [ ] 普攻命中怪物时，只有被击中的那一只顿一下，远处怪物照常移动
- [ ] 闪电链 4 跳，每跳都冻结目标 50ms（不再被冷却合并）
- [ ] 玩家移动不受任何影响（全程全速）
- [ ] 战斗整体感觉更"流畅"而非"一顿一顿"

### 验证方法

```
1. 进游戏 → 观察自动攻击：怪物被击中时只有那一只短暂停顿
2. 选雷电链技能 → 观察弹射：每跳目标依次短暂冻结
3. WASD 移动 → 任何时候都不卡
```

## Phase 4：Travel 拖尾层（~160 行，1.5h）

### 4.1 音频 Travel（SoundManager.ts）

新增 `playTravel()` 私有方法 + 5 个公开包装器：

| 方法 | 原型 | 技术 |
|------|------|------|
| `skillWoodTravel(x)` | 嗖声 | 带通滤波噪声 2000→400Hz，0.5s |
| `skillWaterTravel(x)` | 嘶声+嗖 | 高通噪声 3000Hz + 带通扫频，0.3s |
| `skillPaintTravel(x)` | 嗡声 | 锯齿波振荡器 600→200Hz + 噪声，0.4s |
| `skillWhirlwindTravel(x)` | 嗖声(风) | 带通滤波噪声 3000→600Hz，0.35s |
| `skillLightningTravel(x,y)` | 噼啪声 | 方波脉冲 800→2000Hz + 高通噪声，0.15s |

### 4.2 视觉 Travel（SkillManager.ts + VFX.ts）

利用 Phaser `follow` 参数，投射物创建时挂微型拖尾 Emitter：

```typescript
// 木构加固拖尾示例
const trail = scene.add.particles(x, y, 'px_white', {
  follow: beam,           // 跟随投射物精灵
  quantity: 1,
  frequency: 40,          // 每40ms吐一个粒子
  lifespan: 120,
  scale: { start: 0.3, end: 0 },
  alpha: { start: 0.6, end: 0 },
  tint: [0xc4884d, 0xdaa060, 0x8b6914],
  maxAliveParticles: 8,   // 拖尾不超过8个粒子
});
projectile.trail = trail; // 投射物销毁时清理 trail
```

5 个技能各有拖尾：

| 技能 | 拖尾颜色 | frequency | lifespan | 样式 |
|------|---------|-----------|----------|------|
| wood | 棕色(木屑) | 40ms | 120ms | 方形小点 |
| water | 蓝色(水珠) | 35ms | 100ms | 圆点 |
| paint | 彩虹随机 | 30ms | 150ms | 彩色圆点 |
| whirlwind | 青白(风流) | 20ms | 80ms | 细长线 |
| lightning | 白色(电火花) | 串联线段挂粒子 | — | `lightningArc` 线段中 |

### 验收标准

- [ ] 木梁飞行时身后拖棕色粒子尾巴，同时有"嗖"的下扫频声
- [ ] 水幕弹飞行时身后拖蓝色水珠，同时有"嘶"的湿感声
- [ ] 颜料弹飞行时身后拖彩虹粒子，同时有"嗡"的能量声
- [ ] 旋风刃飞行时身后拖青色流线，同时有"嗖"的风切声
- [ ] 雷电链弹射时线段间挂白色电火花，同时有"噼啪"声
- [ ] 5 个投射物同时飞行无音频卡顿、无帧率下降

### 验证方法

```
1. 每次选技能后观察投射物飞行 → 看是否有拖尾
2. 闭上眼听 → 是否能分辨出正在飞的是哪个技能
3. F12 → Performance → 录制 10 秒战斗 → 确认无 >50ms 帧
```
