# /build — 构建检查

TypeScript 类型检查 + Vite 生产构建，确保代码可发布。

## 触发词

"构建" "打包" "编译" "build" "检查" "tsc"

## 执行流程

1. 执行 `npx tsc --noEmit` 做纯类型检查
2. 如有类型错误，逐条列出：文件、行号、错误信息
3. 执行 `npx vite build`
4. 检查 dist/ 产出：
   - JS 包总大小（与上次对比）
   - 是否超过 2MB 预算
   - 产出文件清单
5. 如有构建错误，定位并报告

## 输出

- ✅/❌ 类型检查结果
- ✅/❌ 构建结果
- 包体积（当前 vs 预算 vs 上次）
- 错误/警告清单
