# 怪物系统优化需求文档

## 1. 现状分析

### 1.1 当前问题

| 问题 | 描述 |
|------|------|
| 怪物数量不足 | 每波产量 1~8 只，无法形成尸潮爽感 |
| 怪物速度过快 | 风蚀怪速度 3.2（192 px/s），玩家难以走位闪避 |
| 怪物血量偏高 | 后期怪物 HP 50~80，击杀节奏拖沓 |
| 生成间隔长 | 前期 4 秒一波，后期 1.1 秒一波，密度不够 |
| 难度曲线不对 | 怪物少→打不到怪→升级慢→技能弱→更打不动怪，恶性循环 |

### 1.2 核心矛盾

当前怪物设计偏向「高血量 + 快速 + 少量」的精英模式，但本游戏定位为**肉鸽割草**，应改为「低血量 + 慢速 + 海量」的尸潮模式。

## 2. 优化目标

- **数量多**：同时在场怪物数 30~80+，屏幕密集但有走位空间
- **移速慢**：怪物移速控制在玩家移速的 30%~60%，让玩家能轻松走位
- **比较弱**：普通怪物 1~3 次攻击即可击杀，前期 1 刀一个
- **玩起来爽**：击杀反馈密集（经验球满地、屏幕震动、音效叠层）

## 3. 具体方案

### 3.1 怪物模板调整

所有怪物的「基础值」应大幅降低，通过「数量」来弥补威胁：

| 怪物 | 当前 HP | 目标 HP | 当前速度 | 目标速度 | 当前伤害 | 目标伤害 |
|------|---------|---------|----------|----------|----------|----------|
| 白蚁怪 | 20 | **8~10** | 2.8 | **1.5~1.8** | 3 | **2** |
| 风蚀怪 | 35 | **12~15** | 3.2 | **2.0~2.2** | 5 | **3** |
| 酸雨怪 | 45 | **15~18** | 1.8 | **1.2~1.5** | 6 | **3** |
| 火焰怪 | 50 | **18~22** | 2.4 | **1.8~2.0** | 8 | **4** |
| 冻融怪 | 80 | **25~30** | 1.4 | **1.0~1.2** | 10 | **5** |

> 速度单位同 config 中的 spec 值（最终乘以 SPEED_FACTOR=60 转为 px/s）

### 3.2 波次系统重构

改为**海量尸潮 + 间歇爆发**的波次节奏：

| 时间段 | 生成间隔 | 每波数量 | 怪物构成 | 说明 |
|--------|----------|----------|----------|------|
| 0~30s | 1500ms | 5~8 | 全白蚁 | 教学期，让玩家感受一刀一个 |
| 30~60s | 1200ms | 8~12 | 白蚁 80% + 风蚀 20% | 开始有小量混合 |
| 60~120s | 1000ms | 12~18 | 白蚁 60% + 风蚀 25% + 酸雨 15% | 数量明显上升 |
| 120~180s | 800ms | 15~22 | 四怪混合 | 进入高潮 |
| 180~240s | 600ms | 18~28 | 五怪混合 | 尸潮高峰 |
| 240~300s | 500ms | 22~35 | 五怪全上 | 终极尸潮 |

> 每波的 countPerWave 应加入随机浮动 ±20%，避免节奏过于整齐。

### 3.3 同时在场怪物上限

- 设置 **最大同时在场数 = 80**
- 达到上限时暂停生成，直到有怪物死亡
- 防止性能崩溃

### 3.4 视觉反馈强化

配合数量提升，需强化击杀反馈：

| 反馈 | 说明 |
|------|------|
| 击杀粒子爆发 | 每只怪物死亡时粒子数量翻倍 |
| 经验球雨 | 经验球掉落更密集，间隔自动拾取范围增大 |
| 屏幕边缘红光 | 当剩余怪物 > 50 时屏幕边缘闪烁红光提示压力 |
| 击杀计数 | HUD 显示累计击杀数，滚数字带来成就感 |
| 音效叠层 | 批量击杀时音效节奏加快 |

### 3.5 平衡性注意

- 怪物血量降低后，**技能伤害暂时不需调整**（击杀更快 = 升级更快 = 更爽）
- 但需注意：怪物数量增加 = 玩家承伤频率增加，需适当增强玩家初始闪避/无敌帧
- 建议玩家增加 0.3s 受伤无敌帧，避免被连续攻击秒杀

## 4. 预期效果

优化后玩家体验预期：

```
开局：白蚁慢慢爬过来 → 一刀一个 → 经验球乱滚
中期：屏幕出现 30+ 怪物 → 技能甩出去一片倒 → 升级选技能 → 更强
后期：80 只怪物同时从四面八方涌来 → 玩家在尸潮中穿梭 → 技能满屏特效
通关：击杀数 500+ → 成就感满满
```

## 5. 开发优先级

| 优先级 | 内容 |
|--------|------|
| P0 | 怪物模板数值调整（血量/速度/伤害） |
| P0 | 波次重构（生成间隔/每波数量） |
| P1 | 同时在场上限 + 暂停生成逻辑 |
| P1 | 玩家受伤无敌帧 0.3s |
| P2 | 击杀反馈强化（粒子/音效） |
| P2 | 击杀计数 HUD |
| P3 | 屏幕边缘红光提示 |
